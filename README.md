## 📁 Project Structure

## 🚀 Getting Started

1. **Clone the repository**

   ```bash
   git clone https://github.com/Aditya4kushwaha/Making-HTTP-Server
   cd Making-HTTP-Server

2. **Install dependencies**
    ```bash
   npm install

4. **Run the server**
   ```bash
   node index.js
